# 038 : パンデジタル倍数\(\*\)

192 に 1, 2, 3 を掛けてみよう.

> 192 × 1 = 192  
> 192 × 2 = 384  
> 192 × 3 = 576

積を連結することで1から9のパンデジタル数 192384576 が得られる. 192384576 を 192 と \(1,2,3\) の連結積と呼ぶ.
