# 028 : 螺旋状に並んだ数の対角線(\*\*)

1から初めて右方向に進み時計回りに数字を増やしていき, 5×5の螺旋が以下のように生成される:

|        |       |       |       |        |
| ------ | ----- | ----- | ----- | ------ |
| **21** | 22    | 23    | 24    | **25** |
| 20     | **7** | 8     | **9** | 10     |
| 19     | 6     | **1** | 2     | 11     |
| 18     | **5** | 4     | **3** | 12     |
| **17** | 16    | 15    | 14    | **13** |

(表のかきかたと赤色文字)

両対角線上の数字の合計は101であることが確かめられる.

$$1001×1001$$の螺旋を同じ方法で生成したとき, 対角線上の数字の和はいくつか?
