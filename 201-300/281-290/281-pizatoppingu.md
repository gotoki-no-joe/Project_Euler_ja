# 281 : ピザトッピング\(\*\)

真円のピザを m×n 個のピースに等分割し、各カットにちょうど 1 つずつトッピングを載せたい。

m \(m ≥ 2\)種の異なるトッピングをのせ、各トッピングがちょうど n \(n ≥ 1\)カットからなるピザを何通り作れるかを f\(m,n\) で表す。  
反転は別物として、回転は同じものとして数える。

つまり例えば、f\(2,1\) = 1, f\(2,2\) = f\(3,1\) = 2, f\(3,2\) = 16 である。f\(3,2\) を下に示す。
