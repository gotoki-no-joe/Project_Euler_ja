# 381 : (素数-k)の階乗

素数$$p$$に対して、$$\displaystyle S(p) = \sum_{k=1}^5 (p-k)! \mod p$$としよう。

例えば p=7 の場合、\
&#x20;(7-1)! + (7-2)! + (7-3)! + (7-4)! + (7-5)! = 6! + 5! + 4! + 3! + 2! = 720+120+24+6+2 = 872\
$$872 \mod 7 = 4$$なので$$S(7) = 4$$である。

$$5 \leq p < 100$$に関して$$\sum S(p) = 480$$となる。

$$5 \leq p < 10^8$$に関して$$\sum S(p)$$を求めよ。
