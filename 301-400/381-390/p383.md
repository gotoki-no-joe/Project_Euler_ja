# 383 : 階乗の整除性比較(\*\*)

$$n$$を$$5^x$$で割り切ることができる最大の整数$$x$$を$$f_5(n)$$で表すとしよう。\
&#x20;例えば$$f_5(625000) = 7$$となる。

$$f_5((2i-1)!) < 2f_5(i!)$$かつ$$1 \leq i \leq n$$を満たす$$i$$の個数を$$T_5(n)$$で表すとしよう。

$$T_5(10^3) = 68, T_5(10^9) = 2408210$$であることが確認できる。

$$T_5(10^{18})$$を求めよ。
