# 397 : 放物線上の三角形

放物線$$y = x^2/k$$上に3点 A$$(a, a^2/k)$$, B$$(b, b^2/k)$$, C$$(c, c^2/k)$$をとる。$$k,a,b,c$$は整数であるとする。

$$F(K,X)$$を、$$1 \leq k \leq K$$かつ$$-X \leq a < b < c \leq X$$で、三角形ABCの少なくともひとつの角が45度となるような組み合わせ$$(k, a, b, c)$$の個数としよう。

例えば$$F(1, 10) = 41, F(10, 100) = 12492$$となる。\
$$F(10^6, 10^9)$$を求めよ。
