# 329 : 素数カエル

Susanは素数カエルを飼っている。

このカエルは、1から500までの番号がつけられた500個のマスの上を跳びまわる。\
左右のいずれかのマスに等確率で跳ぶことだけができ、範囲\[1,500]の外に跳ぶことはできない。\
（いずれかの端に着地したなら、次は自動的に移動可能なマスに跳ぶ。）

素数である数が書かれたマスにいる場合、次のマスに跳ぶ直前に、カエルは2/3の確率で `P` (PRIME)と鳴き、1/3の確率で `N` (NOT PRIME)と鳴く。\
素数でない数が書かれたマスにいる場合、次のマスに跳ぶ直前に、カエルは1/3の確率で `P` と鳴き、2/3の確率で `N` と鳴く。

カエルの開始位置は全てのマスに対し等確率でランダムであり、また最初の15回の鳴き声を聞いたとすると、`PPPPNNPPPNPPNPN` と聞こえる確率は何か。

約分済みの分数 p/q の形で答えよ。
