# 340 : クレイジー関数

整数$$a, b, c$$を定めたとき、**クレイジー関数**$$F(n)$$を次のように定義する：  
$$F(n) = n - c （n > b のとき）$$  
$$F(n) = F(a + F(a + F(a + F(a + n)))) （n ≤ b のとき）$$

また、$$S(a, b, c) = \displaystyle \sum_{n=0}^b F(n)$$と定義する。

例えば,$$a = 50, b = 2000, c = 40$$なら、$$F(0) = 3240, F(2000) = 2040$$である。  
また$$S(50, 2000, 40) = 5204240$$である。
