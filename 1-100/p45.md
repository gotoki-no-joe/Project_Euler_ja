# 045 : 三角数, 五角数, 六角数

三角数, 五角数, 六角数は以下のように生成される.

|  |  |  |
| :--- | :--- | :--- |
| 三角数 | $$T_n=n(n+1)/2$$ | 1, 3, 6, 10, 15, ... |
| 五角数 | $$P_n=n(3n-1)/2$$ | 1, 5, 12, 22, 35, ... |
| 六角数 | $$H_n=n(2n-1)$$ | 1, 6, 15, 28, 45, ... |

$$T_{285} = P_{165} = H_{143} = 40755$$であることが分かる.
