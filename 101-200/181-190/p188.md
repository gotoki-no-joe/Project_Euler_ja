# 188 : ある数のハイパーべき乗

数$$a$$の正整数$$b$$による ハイパーべき乗 \(hyperexponentiation\) または テトレーション \(tetration\) を$$a↑↑b$$または$$^ba$$と書き, 以下のように再帰的に定義する.

* $$a↑↑1 = a$$
* $$a↑↑(k+1) = a^{(a↑↑k)}$$

定義によれば$$3↑↑2 = 3^3 = 27$$であり,$$3↑↑3 = 3^{27} = 7625597484987$$となる. また, $$3↑↑4$$は大体$$10^{3.6383346400240996 \times 10^{12}}$$となる.
