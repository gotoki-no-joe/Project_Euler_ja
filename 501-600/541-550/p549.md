# 549 : 階乗の可除性

10が $$m!$$ を割り切る最小の整数 $$m$$ は5である。\
25が $$m!$$ を割り切る最小の整数 $$m$$ は10である。

ここで、$$s(n)$$ を「$$n$$ が $$m!$$ を割り切る最小の整数 $$m$$」と定義する。\
つまり $$s(10) = 5, s(25) = 10$$ となる。\
また、$$S(n)$$ を2から$$n$$ までの $$s(i)$$ の総和、すなわち $$\sum_{i=2}^n s(i)$$ とする。\
例えば、$$S(100) = 2012$$ である。

このとき、$$S(10^8)$$ を求めよ。
