# 518 : 3つ組素数と等比数列x

以下のような3つ組全てに対し _S_(_n_) = Σ _a_ + _b_ + _c_ としよう:

* _a_, _b_, _c_ は素数.
* _a_ < _b_ < _c_ < _n_
* _a_+1, _b_+1, _c_+1 が等比数列となる

例えば, 以下のような3つ組により S(100) = 1035 となる:

(2, 5, 11), (2, 11, 47), (5, 11, 23), (5, 17, 53), (7, 11, 17), (7, 23, 71), (11, 23, 47), (17, 23, 31), (17, 41, 97), (31, 47, 71), (71, 83, 97)

S(108) を求めよ.
