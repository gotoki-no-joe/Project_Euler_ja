# 511 : 素敵な整除性を持つ数列

以下のような長さ $$n$$ の正整数の数列 $$\{a_i\}_{1 \leq i \leq n}$$ の個数を $$Seq(n,k)$$ としよう：

* $$1 \leq i \leq n$$ に対し $$n$$ が $$a_i$$ で割り切れる
* $$n + a_1 + a_2 + \dots + a_n$$ が $$k$$ で割り切れる

例えば:

Seq(3,4) = 4, そしてその4つの数列は:\
{1, 1, 3}\
{1, 3, 1}\
{3, 1, 1}\
{3, 3, 3}

Seq(4,11) = 8, そしてその8つの数列は:\
{1, 1, 1, 4}\
{1, 1, 4, 1}\
{1, 4, 1, 1}\
{4, 1, 1, 1}\
{2, 2, 2, 1}\
{2, 2, 1, 2}\
{2, 1, 2, 2}\
{1, 2, 2, 2}
