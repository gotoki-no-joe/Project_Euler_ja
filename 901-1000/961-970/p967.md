# 967 : B-三分可能

正の整数 $$n$$ は、$$n$$ の異なる素因数のうち $$B$$ 以下のものの和が 3 で割り切れるとき、 $$B$$ **-三分可能**(B-trivisible)であるという。

例えば、 $$175 = 5^2 \cdot 7$$ は 10-三分可能である。なぜなら $$5 + 7 = 12$$ が3で割り切れるからである。\
同様に、175 は4-三分可能である。なぜなら 175 を割るすべての素数は 4 より大きく、空の和 0 は 3 で割り切れるからである。\
一方、 175 は 6-三分可能ではない。なぜなら関連する素数の和は 5 であり、これは 3 で割り切れないからである。

$$F(N, B)$$ を $$N$$ 以下の $$B$$-三分可能な整数の個数とする。

例えば、 $$F(10, 4) = 5$$ であり、4-三分可能な数は 1,3,5,7,9 である。\
また、$$F(10, 10) = 3$$ および $$F(100, 10) = 41$$ が与えられている。

$$F(10^{18}, 120)$$ を求めよ。
