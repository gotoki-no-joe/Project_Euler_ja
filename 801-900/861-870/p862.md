# 862 : 大きな数字の順列

正の整数 $$n$$ に対し$$T(n)$$ を、 $$n$$ の桁を並べ替えてできる $$n$$ より真に大きい整数の個数と定義する。

先頭に 0 は許されないため、$$n = 2302$$ のときに可能な全ての順列は次のとおり:\
2023, 2032, 2203, 2230, **2302**, 2320, 3022, 3202, 3220

よって $$T(2302) = 4$$ である。

さらに $$S(k)$$ を、すべての $$k$$ 桁の数 $$n$$ についての $$T(n)$$ の総和と定義する。\
$$S(3) = 1701$$ が与えられている。

$$S(12)$$ を求めよ。
