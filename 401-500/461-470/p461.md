# 461 : ほぼπ

すべての非負整数 $$k$$ について $$f_n(k) = e^{k/n} - 1$$ としよう。

驚くべきことに、 $$f_{200}(6)+f_{200}(75)+f_{200}(89)+f_{200}(226)=\underline{3.1415926}44529\cdots\approx\pi$$ である。

実際これは式 $$f_n(a) + f_n(b) + f_n(c) + f_n(d)$$ による  $$n=200$$ における $$\pi$$  の最良近似である。

誤差 $$|f_n(a) + f_n(b) + f_n(c) + f_n(d) - \pi|$$ が最小となる $$a,b,c,d$$ に対し $$g(n)=a^2 + b^2 + c^2 + d^2$$ としよう。（ $$|x|$$ は $$x$$ の絶対値を表す。）

$$g(200)=6^2+75^2+89^2+226^2=64658$$ が与えられている。

$$g(10000)$$ を求めよ。
