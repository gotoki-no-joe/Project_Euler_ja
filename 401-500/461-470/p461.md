# 461 : ほぼπ

任意の非負整数 $$k$$ について $$f_n(k) = e^{k/n} - 1$$ としよう。

驚くべきことに、 $$f_{200}(6)+f_{200}(75)+f_{200}(89)+f_{200}(226)=\underline{3.1415926}44529\cdots\approx\pi$$ である。

実際これは $$n=200$$ における $$f_n(a) + f_n(b) + f_n(c) + f_n(d)$$ の形での $$\pi$$  の最良の近似である。

$$g(n)=a^2 + b^2 + c^2 + d^2$$ とする、ここで $$a,b,c,d$$ は誤差 $$|f_n(a) + f_n(b) + f_n(c) + f_n(d) - \pi|$$ を最小にするものである。（ $$|x|$$ は $$x$$ の絶対値を表す。）

$$g(200)=6^2+75^2+89^2+226^2=64658$$ が与えられている。

$$g(10000)$$ を求めよ。
